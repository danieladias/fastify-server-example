# Fastify Server Example

Experimenting with Fastify

### How to Run

Create a `.env` file specifying in which port to run this server.

```yarn && start:dev```

### HTTP Test

```
GET http://127.0.0.1:3001/api/v1/users/me
Accept: application/json
```
